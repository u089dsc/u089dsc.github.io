一些杂项

第一次使用这玩意儿，随便发个东西试试

图论：

  1. 最短路(dijkstra,floyed,spfa)
  2. 次短路
  3. tarjan 缩点(解决有向图强连通会了，边双、点双还不是很会)
  4. 2-sat 限制选/不选，用Tarjan解决。
  5. 网络流(dinic)，二分图也可用网络流解决

树论：

1. LCA
 2. 树链剖分(重链剖分，长链剖分)
 3. dsu on tree(树上启发式合并)
 4. 最小生成树(kruskal(排序后并查集))
 5. 次小生成树(最小生成树后枚举每一个没有在最小生成树上的边)
 6. 点分治 在查询链信息时考虑

dp: 

1. 普通dp
 2. 数位dp
 3. dp优化
      1. 若维护可行性：bitset优化
      2. 单调队列优化
      3. 斜率优化
      4. 四边形不等式优化(不太会)

数据结构：

1. 线段树、树状数组
 2. 平衡树(fhq)
 3. 主席树(可持久化线段树)
 4. 扫描线
 5. 堆（priority_queue)

根号算法：

1. 莫队
 2. 分块

数论：

1. gcd,exgcd
 2. 组合数

字符串：

1. sam，sa
 2. ~~hash+二分~~

奇淫技巧：

1. 随机哈希

 2. set
